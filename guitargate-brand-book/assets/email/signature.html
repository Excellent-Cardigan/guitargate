How to Deploy This Signature

  • Gmail: Settings → See all settings → General → Signature → New. Paste the raw HTML using the source editor.
  • Apple Mail: Preferences → Signatures. Create new, then edit HTML via Stationery path or use a browser-to-mail copy.
  • Outlook: File → Options → Mail → Signatures. Paste as HTML.
  • The (G) logomark uses a text character (Georgia serif) — if your email client supports it, swap for an embedded PNG of the real (G) mark at 40×40px for sharper rendering.
  • The trueline "Keep it in your hands." uses Georgia italic as a web-font-safe approximation of Transducer Extended Bold italic. This is the correct choice for email (web fonts are stripped by most clients).
  • Test before deploying in: Gmail (web), Apple Mail, Outlook 365, and at least one mobile client.
