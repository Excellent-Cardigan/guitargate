FONT SUBSTITUTION NOTE

All three typefaces require the Adobe Fonts kit nem7lpy to display correctly in browsers. Without it, pages fall back to Inter / system-ui for display and body, and ui-monospace for annotations. To test with real fonts, include <link rel="stylesheet" href="https://use.typekit.net/nem7lpy.css"> in the HTML <head>, or use the @import in styles.css (already included).

In Illustrator: fonts load from your installed Adobe Fonts via Creative Cloud. Before sending any file to print, use Type > Create Outlines to convert all text to paths.
